Bootstrap-4.5- Introduction

Cette page fonctionne avec Bootstrap 4 en CDN

Présentation générale

  • Bootstrap est un framework 'un cadre de travail' CSS "cadriciel"
  • C'est une librairie côté client (front-end)
  • Mise en forme les pages web
  • C'est une librairie CSS 3, il y as également des composants HTML 5 et JavaScript
  • Basé sur Sass. (CSS avancé) et normalize " eric meyer"
  • Comporte également une grille de mmise en page.

Intérêt

  • Uniformisation pour les navigateurs
  • Gain de temps
  • Styles identiques
  • Grille responsive
  • Beaucoup de suppléments en plus
  • Plateforme Gratuite
  • Bonne gestion des medias quieries
